# ShelterTrack Client

## Description
Frontend application for ShelterTrack, a React single-page application for managing shelters and animals.

## Technologies
- React
- React Router DOM
- React Bootstrap
- Bootstrap
- React Icons

## Scripts
- `npm start` - Start development server
- `npm run build` - Create production build
- `npm test` - Run tests

## Notes
- The client is configured to proxy API requests to `http://localhost:8888`.
- The app entry point is `src/index.js`, and routes are defined in `src/App.js`.
