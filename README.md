# ShelterTrack

A MVP web application for managing shelters and animals.

## What It Does
ShelterTrack allows users to:
- View and manage animal shelters
- Register and track animals in shelters
- Filter and sort animals by various criteria
- Edit shelter and animal information
- Move animals between shelters

## Tech Stack

### Frontend
- React (v19.2.4)
- React Router DOM (v7.14.0)
- React Bootstrap (v2.10.10)
- Bootstrap (v5.3.8)
- React Icons

### Backend
- Node.js
- Express.js
