# ShelterTrack Backend

## Description
Backend application for managing shelters and animals.

## Technologies
- Node.js
- Express.js

## API Endpoints

### Shelter
- GET /shelter/list
- POST /shelter/create
- GET /shelter/get
- POST /shelter/update
- POST /shelter/delete

### Pet
- GET /pet/list
- POST /pet/create
- GET /pet/get
- POST /pet/update
- POST /pet/delete

## How to run
- node app.js
